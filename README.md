# 🚀 Dijkstra Algorithm Visualizer

## 📌 Description

This project is an interactive web-based visualization of Dijkstra’s Algorithm where users can dynamically add nodes and edges to create a custom graph. It simulates the shortest path calculation in real-time, showing how the algorithm traverses through nodes step by step. A routing table is also provided to display the final shortest distances and paths.

---

## 🎯 Objectives

* To understand shortest path algorithms through visualization
* To simulate graph-based routing systems
* To demonstrate how routing tables are formed

---

## ⚙️ Features

* ➕ Add custom nodes
* 🔗 Connect nodes using edges with weights
* ▶️ Start simulation of Dijkstra’s Algorithm
* 📍 Visual path traversal
* 📊 Routing table showing shortest paths and distances
* 🔄 Real-time updates

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 🧠 How It Works

1. User creates a graph by adding nodes
2. Edges are added with weights (cost/distance)
3. Algorithm starts from a source node
4. It selects the shortest distance node (greedy approach)
5. Updates distances of neighboring nodes
6. Repeats until all nodes are visited
7. Final shortest path is displayed along with routing table

---

## 📚 Concepts Used

* Dijkstra’s Algorithm

* Greedy Algorithms
* Computer Network Routing
